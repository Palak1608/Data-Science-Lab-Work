# Lab 2 - Control Statements

## Description
This lab focuses on implementing control statements in Python. It includes programs that use conditional statements and loops to control the flow of execution.

## Topics Covered
- if-else statements  
- Conditional logic using operators  
- for loop  
- while loop  
- Basic problem-solving using control structures  

## Programs Implemented
- Check whether a number is even or odd  
- Find the largest of three numbers  
- Print numbers using for loop  
- Print numbers using while loop  
- Calculate sum of first n natural numbers  

## Objective
To understand how decision-making and iteration work in Python programs using control statements.

## Tools Used
- Python  
- Google Colab  

## Conclusion
This lab helps in building logical thinking by using conditions and loops, which are essential for writing efficient programs.
