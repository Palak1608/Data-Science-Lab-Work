# Sentiment Analysis using Machine Learning

## Description
This project performs sentiment analysis on textual data to classify reviews as positive or negative. The objective is to understand how machine learning can be used to analyze public opinion from text data.

## Problem Statement
Analyze text data to determine the sentiment (positive or negative) expressed in reviews. This approach can be extended to social media monitoring for understanding opinions about products, brands, or events.

## Dataset
- Source: IMDB Movie Reviews Dataset
- Description: Contains 50,000 movie reviews labeled as positive or negative sentiment.

## Steps Performed
1. Data Loading and Exploration  
2. Data Preprocessing (cleaning text data)  
3. Feature Extraction using TF-IDF  
4. Train-Test Split  
5. Model Training using Logistic Regression  
6. Model Evaluation (Accuracy, Confusion Matrix, Classification Report)  
7. Predictions on new text samples  

## Results
- Achieved approximately **89% accuracy**
- The model effectively classifies positive and negative sentiments
- Balanced dataset helped improve model performance

## Tools Used
- Python  
- Pandas  
- Matplotlib  
- Scikit-learn  
- Regular Expressions (re)

## Conclusion
This project demonstrates how natural language processing techniques can be used to analyze text data and extract meaningful insights. The same approach can be applied to social media data for real-world sentiment analysis.
