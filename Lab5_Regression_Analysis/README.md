# Lab 5 - Regression Analysis

## Description
This lab focuses on implementing Linear Regression to analyze the relationship between variables and predict outcomes.

## Topics Covered
- Linear Regression  
- Model training using scikit-learn  
- Prediction of values  
- Data visualization using matplotlib  

## Programs Implemented

- Applied Linear Regression model  
- Predicted output values  
- Plotted regression line on graph  

## Objective
To understand how regression models can be used to predict continuous values and analyze relationships between variables.

## Tools Used
- Python  
- Pandas  
- Matplotlib  
- Scikit-learn  

## Conclusion
This lab demonstrates how Linear Regression can be used to model relationships between variables and make predictions based on data.
