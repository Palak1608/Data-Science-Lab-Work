# Student Performance Analysis

## Description
This project analyzes student performance using Python and visualizes key insights.

## Dataset
- Manually created dataset of student marks

## Steps Performed
1. Data Cleaning
2. Exploratory Data Analysis
3. Visualization (bar chart, pie chart, box plot)
4. Grade Classification
5. Prediction using Linear Regression

## Results
- Identified top-performing students
- Analyzed subject-wise performance
- Classified students into grades

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Conclusion
Data analysis helps understand patterns in student performance and improve decision-making.
