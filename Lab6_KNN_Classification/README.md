# Lab 6 - KNN Classification

## Description
This lab focuses on implementing the K-Nearest Neighbors (KNN) algorithm to perform classification tasks based on similarity between data points.

## Topics Covered
- K-Nearest Neighbors (KNN) algorithm  
- Data splitting into training and testing sets  
- Model training  
- Prediction and evaluation  

## Programs Implemented
- Created a dataset with features such as age and salary  
- Applied KNN classifier  
- Predicted class labels  
- Evaluated model accuracy  

## Objective
To understand how KNN works for classification problems by identifying the nearest neighbors and predicting outcomes based on them.

## Tools Used
- Python  
- Pandas  
- Scikit-learn  

## Conclusion
This lab demonstrates how KNN can be used to classify data effectively by analyzing the similarity between data points.
