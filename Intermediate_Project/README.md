# Customer Segmentation using K-Means

## Description
This project performs customer segmentation using K-Means clustering. The goal is to group customers based on their purchasing behavior and identify different customer segments.

## Dataset
- Source: Kaggle (Customer Segmentation Dataset)
- Description: The dataset contains customer information such as Age, Gender, Annual Income, and Spending Score.

## Steps Performed
1. Data Loading and Exploration
2. Data Cleaning (handled missing values)
3. Feature Selection (Age, Annual Income, Spending Score)
4. Feature Scaling using StandardScaler
5. K-Means Clustering (k = 5)
6. Data Visualization (scatter plots and cluster distribution)
7. Cluster Analysis and Interpretation

## Results
- Customers were divided into 5 distinct clusters
- Identified high-value customers (high income & high spending)
- Found potential customers (high income but low spending)
- Observed different spending behaviors across age groups

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Conclusion
Customer segmentation helps businesses understand customer behavior and enables targeted marketing strategies to improve engagement and revenue.
