# Lab 3 - NumPy

## Description
This lab focuses on using NumPy for numerical computations in Python. It introduces array creation and basic operations that are essential for data analysis.

## Topics Covered
- Creating NumPy arrays  
- Array indexing and slicing  
- Mathematical operations on arrays  
- Basic statistical functions  

## Programs Implemented
- Creating arrays using NumPy  
- Performing arithmetic operations on arrays  
- Accessing elements using indexing  
- Finding mean, sum, and other statistics  

## Objective
To understand how NumPy simplifies numerical computations and enables efficient handling of data using arrays.

## Tools Used
- Python  
- NumPy  
- Google Colab  

## Conclusion
This lab provides a foundation for working with numerical data, which is essential for data science and machine learning applications.
