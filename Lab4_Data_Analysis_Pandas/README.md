# Lab 4 - Data Analysis using Pandas

## Description
This lab focuses on performing data analysis using the Pandas library in Python. It involves loading a dataset, exploring its structure, and performing basic statistical operations.

## Topics Covered
- Reading dataset using Pandas  
- Viewing data using head() and tail()  
- Understanding dataset structure  
- Column selection using indexing  
- Basic statistical operations (mean, etc.)  

## Programs Implemented
- Load dataset into DataFrame  
- Display first and last few rows  
- Extract specific columns  
- Calculate mean of a column using Pandas  

## Objective
To understand how to work with real-world datasets using Pandas and perform basic data analysis operations.

## Tools Used
- Python  
- Pandas  
- Google Colab  

## Conclusion
This lab helps in understanding how to manipulate and analyze data efficiently using Pandas, which is a fundamental skill in data science.
